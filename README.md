## ğŸ” Key Insights
- Customers with **month-to-month contracts** churn the most.  
- **High monthly charges** with **low tenure** strongly indicate churn risk.  
- **Bundled services (Internet + Phone + TV)** increase retention.  
- **Fiber optic users** experience slightly higher churn than DSL users.  
- **Customer service duration** correlates with dissatisfaction and churn.
